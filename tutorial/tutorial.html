Para iniciar a aplicação é necessário fazer um login. Para isso, entra-se na página inicial da aplicação.

Aqui, seleciona-se o nome do terapeuta / professor / técnico e a respetiva password. De seguida, clica-se no botão de login para entrar na aplicação.

Ao fazer login é apresentado um ecrã com vários separadores:

  • Sessões a realizar
  • Histórico de Sessões
  • Terapeutas
  • Crianças

Aqui pode-se consultar as sessões que vão ser realizadas (a azul) e as sessões que foram marcadas, mas não realizadas e já passou a data para as quais foram marcas (a vermelho).

Na lista de sessões a marcar, existe alguma informação sobre a sessão:

  • O nome do terapeuta que vai realizar a sessão
  • Data e hora para as quais a sessão está marcada
  • As crianças que vão participar na sessão
  • O tipo de sessão (neste momento, só são suportadas sessões presenciais)
  • O tipo de exercício escolhido para a sessão
  • Botões de manipulação da sessão:
    • Play
    • Editar Sessão
    • Remover Sessão
    • Duplicar Sessão

Este botão vai ser utilizado para iniciar o player da sessão.

Neste botão, remove-se a sessão correspondente. É mostrado um pop-up de confirmação.

Este botão permite duplicar uma sessão já marcada / realizada. É apresentado um pop-up com os dados da sessão e o utilizador preenche apenas a data e a hora a que se vai realizar a nova sessão.

Também é possível adicionar um novo criança através do botão “Adicionar Criança”. Aqui é mostrado um pop-up com os campos a serem preenchidos.

Ao clicar no botão “Adicionar Sessão” que se encontra no separador “Sessões a realizar”, é direcionado para um wizard que permite adicionar uma nova sessão.

A sessão é feita passo a passo, só sendo possível passar ao próximo passo quando os dados são preenchidos corretamente.

Neste separador, escolhe-se o nome do terapeuta que vai realizar a sessão. Por defeito, é escolhido o terapeuta que fez login.

Quando as condições são satisfeitas o botão “Seguinte” fica ativado e é possível continuar para o próximo passo.

Permite escolher as crianças que vão participar na sessão. Tem que se escolher pelo menos uma criança para poder continuar a criar a sessão.

Como se vê na imagem, a caixa de texto da esquerda contém o nome de todas as crianças que foram inseridas na aplicação.

No lado esquerdo, vão-se ter as crianças que vão participar na sessão.

Para isso, tem que se escolher as crianças que estão na lista do lado esquerdo e adicioná-las à lista do lado direito.

Os botões de manipulação são:

  • Adicionar Criança
  • Remover Criança
  • Seleccionar Tudo
  • Remover Selecção

Para adicionar ou remover uma criança, pode-se duplo clicar no nome da mesma e esta é adicionada à lista oposta.

Para selecionar várias crianças, pode-se usar a tecla Shift (permite selecionar várias crianças sem estas estarem seguidas na lista) ou clicar numa criança e arrastar com o rato (seleciona as crianças que estão seguidas na lista).

Também se podem filtrar as crianças pelo nome para ser mais fácil selecioná-las, para isso escreve-se o nome (ou parte dele) na caixa de texto que fica situada em cima da lista das crianças existentes.

Quando existir pelo menos uma criança na lista de crianças da sessão, o botão seguinte vai ser ativado e pode-se seguir para o próximo passo.
